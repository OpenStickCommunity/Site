# Add-Ons Configuration

This section is for custom add-ons that can be enabled to expand the functionality of GP2040-CE. Due to the large number of add-ons created by the community, each add-on has its own separate documentation page.

:::caution

Some of these add-ons are experimental and not all add-ons are interoperable with all other add-ons.

:::

## Deprecated Add-ons

- `Player Number` - Replaced by the ability to set PID/VID in main (Settings)[../../web-configurator/menu-pages/01-settings.mdx] page.
- `PS4 Mode` - Moved to the (Settings)[../../web-configurator/menu-pages/01-settings.mdx] page under PS4 in the `Current Input Mode`.
- `PS Passthrough` - Moved to the (Settings)[../../web-configurator/menu-pages/01-settings.mdx] page under PS4 in the `Current Input Mode`.
- `Xbox One Passthrough` - Moved to the (Settings)[../../web-configurator/menu-pages/01-settings.mdx] page under `Xbox One` in the `Current Input Mode`.
- `Input History` - Moved to the (Display Configuration)[../../web-configurator/menu-pages/07-display-Configuration.mdx] page under `Layout Options`.
