# Documentation Style Guide

## Key Terms

## Components

### Paths

Use absolute links for components

### Hotkeys

Use following component with multiple buttons as an array.

Labels should always be GP2040-CE labels rather than specific consoles' inputs.

```jsx
<Hotkey buttons={["Button1", "Button2", "Button3"]} />
```

## Linking

Use relative links
