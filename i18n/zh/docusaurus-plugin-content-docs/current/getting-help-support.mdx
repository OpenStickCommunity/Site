# 获取帮助/支持

## Discord

如果您需要帮助，最快获取支持的地方是我们的[Discord服务器](https://discord.gg/k2pxhke7q8)。请首先阅读[置顶帖子](https://discord.com/channels/1049366310389289001/1132450634243772519)，然后在[支持频道](https://discord.com/channels/1049366310389289001/1132447732662018089)创建帖子，并提供以下信息：

1. 您使用的设备名称是什么？
2. 您在哪里买的？请提供销售页面的链接。
3. 您当前运行的固件版本是多少？通过Web配置器查看（例如0.4.3, 0.5.0, 0.7.4等）。
4. 您当前运行的是哪个固件分发版本？提供准确的文件名（例如GP2040-CE_X.X.X_Pico.uf2）。
5. 您遇到了什么问题？
6. 您已经进行了哪些故障排除步骤？

如果这是自定义设备或硬件问题，或者您认为固件问题可能由硬件引起，请包含任何相关图片。

## Github问题

您可以在[Github上打开一个issue](https://github.com/OpenStickCommunity/GP2040-CE/issues/new/choose)。
